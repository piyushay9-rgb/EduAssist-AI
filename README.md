# EduAssist AI

## SDG Goal
SDG 4 - Quality Education

## Project Description
EduAssist AI is a chatbot designed to help students access information about admissions, fees, library timings, timetables, and campus events.

## Features
- Admission Information
- Fee Information
- Library Timings
- Timetable Assistance
- Event Information
- Fallback Response

## Technology Stack
- Python
- Tkinter
- Visual Studio Code
